Bounty Liquidity Guard

A Hedera Agent Kit plugin that rejects illiquid rewards, prompt-leak traps, symbolic bounty bait, and sub-$10 work before an agent spends time on it.

GitHub repo Hedera bounty

Agent Kit Tools

analyze_bounty_liquidity

Runs the full intake gate and emits a decision packet.

check_token_market

Normalizes token market evidence for liquid-value checks.

score_opportunity

Computes expected liquid dollars per hour after competition.

write_hedera_audit_packet

Creates a dry-run digest or writes the packet to Hedera HCS.

Control Layer

The public demo holds no user funds and performs no transfers. It is an enterprise control layer for deciding which online paid-work prospects are credible enough for agents to pursue, with optional Hedera Consensus Service audit logging.

Built with TypeScript, Vitest, `@hashgraph/hedera-agent-kit`, and `@hiero-ledger/sdk`.
